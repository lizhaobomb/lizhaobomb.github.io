## [Hexo官网](https://hexo.io)

### 环境依赖

* [Node.js](https://nodejs.org/en/)

* [git](https://git-scm.com) 

如果这两个环境都已经装好了直接：



快速开始

```
npm install -g hexo-cli

hexo init blog

cd blog

npm install

hexo server

```
